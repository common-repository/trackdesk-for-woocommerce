== Installation ==

1. Install easily with the WordPress plugin control panel or manually download the plugin and upload the folder `trackdesk-for-woocommerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Fill required fields and enable plugin functionality in the settings (Admin > WooCommerce > Trackdesk)
